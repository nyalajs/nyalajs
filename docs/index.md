## Packages

Nyala is distributed as focused npm packages. Install only what you need:

- **[@nyalajs/cli](https://www.npmjs.com/package/@nyalajs/cli)** - Command-line interface
- **[@nyalajs/core](https://www.npmjs.com/package/@nyalajs/core)** - Framework core with DI
- **[@nyalajs/http](https://www.npmjs.com/package/@nyalajs/http)** - HTTP server and routing
- **[@nyalajs/database](https://www.npmjs.com/package/@nyalajs/database)** - ORM and migrations
- **[@nyalajs/validation](https://www.npmjs.com/package/@nyalajs/validation)** - Request validation
- **[@nyalajs/security](https://www.npmjs.com/package/@nyalajs/security)** - Auth & authorization
- **[@nyalajs/tenancy](https://www.npmjs.com/package/@nyalajs/tenancy)** - Multi-tenant support
- **[@nyalajs/cache](https://www.npmjs.com/package/@nyalajs/cache)** - Caching layer
- **[@nyalajs/queue](https://www.npmjs.com/package/@nyalajs/queue)** - Background jobs
- **[@nyalajs/mail](https://www.npmjs.com/package/@nyalajs/mail)** - Email sending
- **[@nyalajs/storage](https://www.npmjs.com/package/@nyalajs/storage)** - File storage
- **[@nyalajs/events](https://www.npmjs.com/package/@nyalajs/events)** - Event system
- **[@nyalajs/config](https://www.npmjs.com/package/@nyalajs/config)** - Configuration
- **[@nyalajs/observability](https://www.npmjs.com/package/@nyalajs/observability)** - Logging
- **[@nyalajs/testing](https://www.npmjs.com/package/@nyalajs/testing)** - Testing utilities
